BOOKS(1)

NAME
	books - list matching results in file

SYNOPSIS
	books [-h | --help] [<option> <string>... <file>]

DESCRIPTION
	Lists all results from a CSV <file> matching <string> according to <option>. Default lists all books.

	-t, --title[=SORT]
		Search for <string> in titles within <file> and list in alphabetical order by title; SORT can 
		be 'title' or 'year'
	
	-a, --author
		Search for <string> in author names within <file> and list in alphabetical order by surname

	-h, --help
		Prints usage statement
	
	-y, --year
		Search for years within <file> in the range from the first <string> to the second <string> and 
		list in order by year
