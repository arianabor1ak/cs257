BOOKS(1)

NAME
	books - list matching results in file

SYNOPSIS
	python3 books.py [-h | --help] [<option> [string]... [sort]]

DESCRIPTION
	Lists all results from a books CSV file matching a case-insensitive <string> according to <option>. Indicating no option
	returns all books.

	-t, --title [sort]
		Search for <string> in titles within books file and list in alphabetical order by title; sort can
		be 'title' or 'year'

	-a, --author
		Search for <string> in author names within books file and list in alphabetical order by surname

	-h, --help
		Prints usage statement

	-y, --year
		Search for books by years within books file in the range from the first <string> to the second <string> and
		list in order by year, inclusive. Use None for either or both strings to have no limit for the start or end year.
